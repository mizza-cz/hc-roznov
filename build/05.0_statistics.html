Vysvětlivky sloupců

Brankáři

# - číslo dresu v posledním zápase (pro případ, že by se měnilo)
Z - počet zápasů, v nichž hráč nastoupil
Min - počet odchytaných minut
Stř - počet střel vyslaných na brankáře
Ink - počet inkasovaných gólů
Zás - počet zásahů
Prů - průměr gólů na zápas ( = inkasované góly / minuty/60 )
Úsp - úspěšnost zásahů ( = zásahy / střely )
SO - shut-outy, vychytané nuly
A - asistence
TM - trestné minuty
